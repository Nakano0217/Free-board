# 黒地に白ペンのお絵描きページ

ローカルで開いて使える簡単なお絵描きページです。特徴:

- 黒背景のキャンバス
- 白ペンで描画
- ペン／消しゴムの切り替え（右上ボタン）
- 全削除ボタン（右上）
- 太さ調整バー（右上）

使い方:

1. このフォルダ内の `index.html` をブラウザで開いてください。
2. 右上の「ペン」ボタンを押すとペン／消しゴムが切り替わります。
3. 「全削除」でキャンバスをクリアできます。
4. スライダーで線の太さを調整できます。
5. キーボードショートカット: `E` で消しゴム切替、`C` で全削除。

備考:

- `index.html`, `styles.css`, `script.js` の3ファイルで実装しています。
- キャンバスはウィンドウサイズに合わせてリサイズしますが、大きなサイズ変更では既存の描画が期待通り保存されない場合があります（簡易的な実装のため）。
